利用規約

最終更新日: 2025年12月7日

第1条（適用）

本規約は、どこでもバスガイド（以下「本アプリ」）の利用に関する条件を定めるものです。 本アプリを利用するすべてのユーザー（以下「ユーザー」）は、本規約に同意したものとみなされます。

第2条（サービスの内容）

本アプリは、以下のサービスを提供します。

  1. 位置情報に基づいた観光ガイド情報の提供
  2. AIペルソナとの会話機能
  3. 訪問履歴の記録と地図表示
  4. 音声読み上げ機能
  5. 関連情報へのリンク提供

第3条（利用条件）

ユーザーは、以下の条件を満たす必要があります。

  • 本規約およびプライバシーポリシーに同意すること
  • 13歳以上であること（13歳未満の場合は保護者の同意が必要）
  • 位置情報の使用を許可すること
  • インターネット接続環境を有すること

第4条（禁止事項）

ユーザーは、以下の行為を行ってはなりません。

  • 本アプリの運営を妨害する行為
  • 本アプリを不正な目的で使用する行為
  • 本アプリのリバースエンジニアリング、逆コンパイル、逆アセンブル
  • 本アプリの知的財産権を侵害する行為
  • 第三者に不利益、損害を与える行為
  • 法令または公序良俗に違反する行為

第5条（位置情報の利用）

  1. 本アプリは、観光ガイド機能を提供するために位置情報を使用します。
  2. 位置情報は、ユーザーのデバイスのローカルストレージに保存されます。
  3. 位置情報の収集と使用の詳細については、プライバシーポリシーをご確認ください。
  4. ユーザーは、いつでもデバイスの設定から位置情報の使用を無効にできます。

第6条（AIガイドの提供情報について）

  1. 本アプリのAIガイドが提供する情報は、AI技術により自動生成されるものです。
  2. 提供される情報の正確性、完全性、有用性について、当方は一切保証しません。
  3. ユーザーは、AIガイドの情報を参考情報として使用し、重要な判断をする際は他の情報源も確認することを推奨します。
  4. AIガイドの情報に基づいて生じたいかなる損害についても、当方は責任を負いません。

第7条（知的財産権）

  1. 本アプリに関する知的財産権は、すべて当方または正当な権利者に帰属します。
  2. ユーザーは、本アプリを個人的かつ非商業的な目的でのみ使用できます。
  3. 本規約に基づく本アプリの使用許諾は、非独占的で譲渡不可能なライセンスです。

  4. 地図データ、外部リンク先の内容については、当方は責任を負いません。
